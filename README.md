## git 学习
```
git add ./xxx
git commit -m "这里写些说明"
git status  查看状态
查看日志： git log
直接放到仓库： git commit --all -m "这是说明"
```
### 版本回退

reset：重置  
--hard： 把代码拿过来  
Head~0： 指向最新的版本   Head~1： 表示更旧的一个版本    
`git reset --hard Head~0`  
也可以 `git reset --hard xxxxxxxx`      xxxxxxxx： 8位的标识符
